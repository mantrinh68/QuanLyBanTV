# Phần mềm Quản lý Bán Tivi

## Giới thiệu
Phần mềm được xây dựng bằng **C# WinForms** theo **mô hình 3 lớp**, giúp quản lý:
- Sản phẩm, khách hàng, nhân viên
- Hóa đơn, doanh thu, thống kê

## Công cụ
- Ngôn ngữ: C# (.NET Framework)
- Giao diện: WinForms
- CSDL: SQL Server
- IDE: Visual Studio 2022
